# Farming Income Tracker

A mobile-friendly app for tractor and vehicle owners to track work, manage farmers, and calculate income automatically.

## Run locally

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   `npm run dev`
